## **Project Description**

The Console Phone Book is a command-line application written in C for managing personal contact information.
It demonstrates core concepts of **imperative programming**, including structures, arrays, file operations, and user input handling.

The program allows users to add, remove, search, sort, and save contacts in a simple and efficient manner.
